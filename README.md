# Week 1 - Basic Sentiment Classifier

## Project
Product Review Sentiment & Insight Generator

## Objective
Build a basic sentiment classifier that predicts whether a product review is Positive, Negative, or Neutral.

## Tools Used
- Python
- TextBlob

## Features
- Accepts user review as input
- Calculates sentiment polarity
- Classifies review into:
  - Positive
  - Negative
  - Neutral

## Example

Input:
This phone is excellent.

Output:
Positive
