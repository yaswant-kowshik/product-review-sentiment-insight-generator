# Week 3 - Multimodal Model (Text + Product Image)

## Objective
Develop a simple multimodal sentiment analysis system that combines product review text with the corresponding product image.

## Technologies Used
- Python
- Pandas
- Pillow
- TextBlob
- Matplotlib

## Features
- Reads product reviews from a CSV file.
- Loads the corresponding product image.
- Performs sentiment analysis using TextBlob.
- Displays the product image along with the predicted sentiment.
