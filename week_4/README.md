# Week 4 - Deep Aspect-Based Sentiment Analysis using BERT

## Objective
Implement aspect-based sentiment analysis using a pre-trained BERT model to determine the sentiment of product reviews.

## Technologies Used
- Python
- Transformers (Hugging Face)
- PyTorch
- Pandas

## Features
- Uses a pre-trained BERT sentiment analysis model.
- Reads reviews and aspects from a CSV file.
- Predicts sentiment for each review.
- Displays confidence scores for each prediction.
