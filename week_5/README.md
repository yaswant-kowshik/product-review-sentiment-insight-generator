# Week 5 - LLM-Powered Review Summarizer & Auto-Responder

## Objective
Use a pre-trained Large Language Model (LLM) to summarize customer reviews and automatically generate a response.

## Technologies Used
- Python
- Hugging Face Transformers
- PyTorch

## Features
- Reads customer reviews from a text file.
- Generates a concise summary using a pre-trained summarization model.
- Produces an automatic customer response.
